Detailed Description

A class to hold meta data that can be generated on the fly.

What we mean by "on the fly"? It means you can define variable name and store it in the data w/o defining a dedicated data container class.

The supported variable type for storage includes double, int, std::string, bool, and vector of those types. The variable is stored as a pair of a specific std::string key and value, both provided by a user. The string is then used to retrieve a variable later. Note that each retrieval requires std::string comparison, hence is somewhat expensive (i.e. takes time). But there are use-cases where speed may not be too important.
