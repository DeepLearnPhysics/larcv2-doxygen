Detailed Description

Analysis event-loop driver that executes larcv::ProcessBase inherit "process modules".

It uses larcv::IOManager behind the scene to run the event loop.
Responsible for propagating a structured configuration parameter sets to configure each process module.
The configuration of larcv::ProcessDriver itself can contain larcv::IOManager configuration.
