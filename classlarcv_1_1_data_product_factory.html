Detailed Description

Factory class for instantiating event product instance by larcv::IOManager This factory class can instantiate a specified product instance w/ provided producer name.


The actual factory core (to which each product class must register creation factory instance) is
a static std::map. Use static method to get a static instance (larcv::DataProductFactory::get)
to access a factory.
