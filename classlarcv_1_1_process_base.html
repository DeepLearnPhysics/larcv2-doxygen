Detailed Description

A base class for "process module" to be run by larcv::ProcessDriver Inherited class must implment 4 pure virtual functions that are called by larcv::ProcessDriver instance.

User defined class PyTensor3DMaker ...

User defined class PyImageStitcher ...

User defined class PyImageMaker ...

User defined class PyCluster3DMaker ...


ProcessBase::configure(const larcv::PSet) is called first with the argument passing the configuration parameters.
ProcessBase::initialize() is called after configure. This is where you may want to initialize variables.
ProcessBase::process(larcv::IOManager&) is called for every event. The argument provides an access to event data.
ProcessBase::finalize() is called after larcv::ProcessDriver finished looping over all events.
